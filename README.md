# Space-Invaders

This is a classic arcade game of Space Invaders using Java Swing, testing with JUnit platform.

### Systen Requirement

Gradle and JDK8

### Play

Run App.java

Control movement of the tank via left and right arrow key, shoot the invaders by clicking space bar.
